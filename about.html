My Tools

You can see my mastery of tools by hovering your mouse over the icons!

IntelliJ

I've been using the IntelliJ IDE since January 2022. The IDE is rather more complex than Eclipse, but it offers more possibilities, such as the plugins found on IntelliJ that are not available on Eclipse. IntelliJ is also cleaner and more ergonomic than its competitor.

DataGrip

I've been using the DataGrip IDE since September 2023. In my opinion, the IDE is easier than the others. It's still very complicated to understand some of the specifics, but I like it nonetheless.

PyCharm

I've been using the PyCharm IDE since September 2022. Following problems with my VS Code IDE, which have now been resolved, I had to find an alternative for my Python projects. PyCharm remains by far my favorite IDE for Python.

Clion

I've been using the Clion IDE since March 2022. I find it an interesting IDE for C++. It has a built-in compiler, so you don't have to find one and try to get it to work, like VS Code, for example. CLion is easy to get to grips with - all that's left is the language!

Fleet

Fleet is an IDE I'm really looking forward to, because of its design and the way it stands out from other IDEs. For the moment, it's in Closed Beta under Whitelist. I can't wait to be selected to test it or for it to be released.

Visual Studio Code

I've been using the Visual Studio Code ide since 2017. It's an IDE known for its ease of use and efficiency. Although I prefer to use dedicated IDEs for certain languages, VS Code remains a very good IDE.

VS Codium

I've been using VS Codium since January 2022. It's an IDE I knew from my first year at the IUT Informatique de Limoges for PostgreSQL. It's easy to learn, but I don't use it every day.

Docker

I've been using Docker since September 2021. I knew it by name and really started using it for various projects. I find it essential for certain things, although I haven't mastered it perfectly.

GitKraken

I've been using GitKraken since January 2022. I wasn't leaning towards it, but now that I've learned about the importance of commits for projects, I've started using it since September 2023.
